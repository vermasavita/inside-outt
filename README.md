# [inside-outt](https://lw0fh.csb.app/)

## About
This is an emoji interpreter. User can put the emoji in input bar and get meaning of the emoji as a output Or user can directly click to the emoji and get the meaning of it.

## Who can use it?
We often struggle with understanding the emojis. Do you too? If yes, then see this library of emojis with meaning.
